Multilingual READMEs use a flag switcher line at the top of each file. The active language is unlinked, the other is a link. Pattern:

```markdown
🇬🇧 English | [🇨🇿 Česky](README.cs.md)
```

```markdown
[🇬🇧 English](README.md) | 🇨🇿 Česky
```

Primary README is `README.md` (English). Translations use `README.{lang}.md` with ISO 639-1 codes (`cs`, `de`, `fr`). The switcher line sits above the H1 heading, separated by a blank line.
